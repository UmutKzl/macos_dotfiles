# dots

> [!WARNING]
> I don't work on this project. If you want to customize your macOS, you can use [mmosga](https://github.com/UmutKzl/mmosga).

## Installation

```bash
brew install stow git
git clone https://github.com/UmutKzl/macos_dotfiles.git ~/dots
cd ~/dots
brew bundle
stow configs_here
```
